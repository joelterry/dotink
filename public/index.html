The Ink language

This is my technical blog and website about a programming language I created called Ink, a functional language that takes after modern JavaScript and Go. I use Ink to do most of my day-to-day scripting and build some of my core apps with it. This website is served by a server written in Ink.

I write here about programming languages, interpreters and compilers, frameworks, web applications, software architecture, and other things. You can read them here. This website is also home to the Ink programming language documentation, which is a work in progress.

If you want to see other stuff I’m up to, you can find more on my website at thesephist.com and on Twitter at @thesephist.
